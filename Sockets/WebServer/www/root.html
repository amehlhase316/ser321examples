You can make the following GET requests

    			
  • /file/sample.html -- returns the content of the file sample.html
    • 
			
  • /json -- returns a json of the /random request
    • 
			
  • /random -- returns index.html
    • 
			
  • /multiply -- returns product of integers num1 and num2
    • 
			
  • /github/users/[user]/repos -- returns info on public repos for [user]
    • 
			
  • /location -- Saves name and location as Json pair and writes to file. **This is publicly viewable! Keep it vague and appropriate. Users may enter name as 'erase' to delete contents of file**
    • 
			http://52.23.163.30/location?name=Shea&location=Oregon --This adds JSON pair name: Shea and location: Oregon to the studentLocations file.
			http://52.23.163.30/location?name=erase -- This overwrites the contents of studentLocations file with blanks.
